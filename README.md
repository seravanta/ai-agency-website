# AI Workflow Agency — Sales Website

One-page sales site for AI Workflow Agency, targeting small businesses in Adelaide, Australia.

## Contents

- `index.html` — single-file website, no build step or framework required

## Sections

1. Hero — headline, subheadline, CTA
2. Problem — pain points (missed leads, slow follow-up, admin overload)
3. Solution — what we build and how it helps
4. How It Works — 3-step process
5. Pricing — Starter / Growth / Full System tiers
6. Testimonials — Adelaide business owner quotes
7. FAQ — accordion with 5 common questions
8. Final CTA — closing statement and button
9. Footer — contact and copyright

## Deployment

This is a static HTML file. Deploy anywhere that serves static files:

- **GitHub Pages** — push to a repo, enable Pages in Settings → Pages → Deploy from branch
- **Netlify** — drag and drop the `website/` folder into the Netlify dashboard
- **Vercel** — import the repo, set root directory to `website/`
- **Railway** — add as a static service pointing to this folder

## Local Preview

Open `index.html` directly in any browser — no server required.
